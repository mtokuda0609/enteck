# enteck
ニッチな商品を売買できるプラットフォーム

# Function list
* 商品出品機能
   - 画像アップロード機能
* 売上一覧照会機能
   - CSV出力機能
* 商品検索機能
* 商品購買機能
* 購買履歴参照機能
* ユーザ登録機能
* ログイン機能

# Technical information
* App
  - Ruby: ruby 2.6.0
  - Ruby on Rails: 5.0.6
  - jQuery
  - Bootstrap
* DB
  - MySQL: 5.5.62
* OS
  - Amazon Linux AMI: 2018.03
* Deploy
  - Heroku
* Version management
  - GitHub
* Debug
  - pry-byebug
